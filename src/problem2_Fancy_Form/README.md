## 📋 About
Interactive React form built with Vite, featuring modern UI components and form validation.

## 🛠️ Tech Stack
- React 18
- TypeScript
- Vite
- CSS Modules
- Deployed on Netlify

## 🏃‍♂️ Quick Start
```bash
pnpm install
pnpm dev
```

## 🌐 Deployment
```bash
pnpm build
netlify deploy --prod --dir=dist
```
